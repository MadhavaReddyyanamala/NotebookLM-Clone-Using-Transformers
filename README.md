# NotebookLM-Clone-Using-Transformers
A document-grounded AI research assistant built with open-source transformer models — replicating the core intelligence of Google's NotebookLM.

## Project Overview

This project implements an abstractive text summarization system using a pretrained transformer model from the Hugging Face Transformers library. The model is fine-tuned on a custom dataset containing text-summary pairs to generate concise and meaningful summaries from longer textual content.
The system leverages state-of-the-art sequence-to-sequence (Seq2Seq) transformer architectures to understand the context of the input text and produce human-like summaries.
